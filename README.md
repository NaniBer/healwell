# HealWell

A Telegram Mini App to help people navigate breakups with AI-powered personalized recovery plans.

## Project Structure

```
healwell/
├── frontend/          # React + Telegram WebApp API
│   ├── src/
│   ├── public/
│   └── package.json
├── backend/           # Node.js + Express API
│   ├── src/
│   │   ├── routes/
│   │   ├── controllers/
│   │   ├── services/
│   │   └── models/
│   └── package.json
└── README.md
```

## Tech Stack

- **Frontend**: React, Telegram WebApp API, Tailwind CSS
- **Backend**: Node.js, Express
- **AI**: OpenAI API / Anthropic Claude API
- **Database**: PostgreSQL

## Getting Started

### Backend Setup

```bash
cd backend
npm install
```

### Frontend Setup

```bash
cd frontend
npm install
```
